
# Commerce Project for CS50

This is project #4 made for CS50 course, an eBay-like e-commerce auction site that will allow users to post auction listings, place bids on listings, comment on those listings, and add listings to a â€œwatchlist.


## Run Locally

Clone the project

```bash
  $ git clone https://github.com/mariante0dorescu/CS50/
```

Go to the project directory

```bash
  $ cd Project2_V2  
```

Create a virtual environment to install dependencies in and activate it:

```bash
  $ virtualenv2 --no-site-packages env
  $ source env/bin/activate
```

Start the server

```bash
  $ python3 manage.py runserver
```
And navigate to http://127.0.0.1:8000/
